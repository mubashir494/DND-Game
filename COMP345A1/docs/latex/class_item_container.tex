\doxysubsection{Detailed Description}
A container class for storing items. 

\doxylink{class_item_container}{Item\+Container} can hold any item type (e.\+g., \doxylink{class_helmet}{Helmet}, \doxylink{class_armor}{Armor}) and is used for various in-\/game storage units like backpacks, chests, etc. It demonstrates polymorphism by managing a collection of \doxylink{class_item}{Item} pointers.
