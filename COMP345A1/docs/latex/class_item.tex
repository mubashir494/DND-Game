\doxysubsection{Detailed Description}
Base class for all item types in the game. 

Game Rules\+: Items can influence character stats and have enchantments. Design\+: Serves as a base for derived item classes like \doxylink{class_helmet}{Helmet}, \doxylink{class_armor}{Armor}, etc. Libraries\+: $<$string$>$ for item names
