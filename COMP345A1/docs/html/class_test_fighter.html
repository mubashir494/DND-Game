Detailed Description

Test Class for the Fighter class It must be a subclass of CppUnit::TestFixture It then uses CPPUNIT_TEST_SUITE() to create the test suite, and CPPUNIT_TEST() to include the test methods in the test suite. CPPUNIT_TEST_SUITE_REGISTRATION() is then used to register the test class in the test registry.
